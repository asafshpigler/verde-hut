### Run Locally

To run **Verde Hut** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/Verde Hut.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/Verde Hut.git
```
